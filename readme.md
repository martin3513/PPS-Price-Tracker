# 📊 조달청 물품 가격 변동 추적기 (PPS Price Tracker)

**나라장터 종합쇼핑몰**의 물품 목록 텍스트 데이터를 파싱하여 엑셀(CSV) 형태로 변환하고, 날짜별 가격 변동 내역을 누적 관리할 수 있는 **웹 기반 도구**입니다.

별도의 서버 설치나 프로그램 다운로드 없이, **HTML 파일 하나만으로** 브라우저에서 즉시 실행이 가능합니다.

## 🌟 주요 기능 (Features)

*   **간편한 실행**: 설치 없이 `index.html` 파일만 열면 바로 사용 가능
*   **비정형 데이터 파싱**: 조달청 웹사이트에서 복사(`Ctrl+A`, `Ctrl+C`)한 텍스트 데이터를 구조화된 표(Grid)로 자동 변환
*   **가격 변동 추적**:
    *   이전 데이터를 업로드하여 신규 데이터와 비교
    *   **인상(🔺), 인하(🔻), 신규(🆕), 동일(-)** 상태 자동 시각화
    *   날짜별 가격 히스토리 누적 관리 (시계열 데이터)
*   **강력한 필터링**:
    *   특정 업체만 모아보기
    *   가격 변동이 발생한 제품만 필터링
    *   최신 날짜 기준 가격 오름차순 정렬
*   **데이터 소유권**: CSV 파일로 데이터를 내보내고 가져오며, 모든 데이터는 사용자 PC에 저장됩니다.

## ⚠️ 주의 사항 (Caution)

이 도구는 조달청 웹사이트의 텍스트 **형식(Format)**에 의존하여 데이터를 분석합니다.
따라서 다음과 같은 경우 데이터가 올바르게 인식되지 않을 수 있습니다:

1.  **복사 방식**: 반드시 페이지 전체를 선택(`Ctrl+A`)한 후 복사(`Ctrl+C`)하여 붙여넣어 주세요. 일부만 드래그하여 복사하면 줄바꿈 형태가 달라져 파싱에 실패할 수 있습니다.
2.  **웹사이트 변경**: 조달청 나라장터 쇼핑몰의 화면 구성이 대폭 변경될 경우, 이 도구의 파싱 로직이 작동하지 않을 수 있습니다.

## 🚀 시작하기 (Getting Started)

1.  이 저장소를 다운로드합니다.
2.  `index.html` 파일을 더블 클릭하여 웹 브라우저(Chrome, Edge 등)에서 실행합니다.

## 📖 사용 방법 (Usage)

### 1. 이전 히스토리 불러오기 (선택 사항)
*   이전에 저장해둔 `CSV` 파일이 있다면, **"1. 이전 히스토리 파일"** 영역에 업로드합니다.
*   과거의 가격 데이터가 로드되어 오늘 데이터와 비교할 준비가 됩니다.

### 2. 신규 데이터 입력
*   **품목 선택**: 데스크톱, 노트북 등 품목을 선택합니다.
*   **날짜 선택**: 기준이 될 날짜를 확인합니다 (기본값: 오늘).
*   **데이터 붙여넣기**: 조달청 목록 페이지에서 `Ctrl+A`로 전체 선택 후 복사한 텍스트를 붙여넣습니다.

### 3. 분석 및 다운로드
*   **[데이터 분석 및 병합]** 버튼을 클릭합니다.
*   하단 표에서 가격 변동 내역(🔺, 🔻)을 확인합니다.
*   **[💾 엑셀(CSV) 다운로드]** 버튼을 눌러 상태를 저장합니다. 이 파일은 다음번에 "이전 히스토리"로 재사용됩니다.
